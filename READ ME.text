Pokédex


Esta Pokédex es una aplicación web que utiliza la Poke API para mostrar información detallada sobre diferentes especies de Pokémon. El diseño está inspirado en el Rotom Phone y busca ofrecer una experiencia divertida y accesible para los fans de la franquicia.

Características

Lista de todos los Pokémon disponibles en la Poke API
Búsqueda por nombre de Pokémon
Detalles de cada Pokémon, incluyendo su tipo, habilidades, estadísticas y más
Diseño intuitivo e innovador inspirado en el Rotom Phone
Uso
Para utilizar esta Pokédex, simplemente accede a la página web y navega por las diferentes secciones. Puedes buscar un Pokémon específico utilizando la barra de búsqueda o navegar por la lista completa de especies. Para obtener más información sobre un Pokémon, haz clic en su tarjeta para ver su página de detalles.

Instalación

No es necesario instalar nada para utilizar esta Pokédex, ya que se trata de una aplicación web. Sin embargo, si deseas descargar y ejecutar el código localmente, puedes hacer lo siguiente:

Clona el repositorio en tu máquina local: git clone https://github.com/usuario/repo.git
Abre el archivo index.html en tu navegador web.
